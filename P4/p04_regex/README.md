# Objetivo Principal de la práctica

Introducir por comando un fichero escrito en c++, y reconocer las siguientes estruxturas del lenguaje:

- Declaración de variables -> int y double, en una única línea
- Utilizaciíon de bucles -> for y while, se almacena tipo de bucle y línea
- Existencia de una función main()
- Comentario (//) o (/**/) y cantidad de cada tipo
